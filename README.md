# 🏠 RentScout AI

AI-powered rental and budget assistant for newcomers in Vancouver.

## Features

- 🤖 AI rental assistant
- 💰 Interactive budget planner
- 📊 Financial dashboard
- 🧠 AI budget insights
- 🏙 Neighborhood guidance
- ⚠ Scam awareness support

## Tech Stack

- Python
- Streamlit
- OpenAI API
- python-dotenv

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
